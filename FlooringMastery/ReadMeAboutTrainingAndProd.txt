Instructions:
Here is how to go about changing modes between "training" and "prod". Go to the .txt file
labeled "Modes.txt". Type into the first line either "training" or "production". If you select
training, none of the data used will be saved to permenant storage. if you type in "production"
the program will work as normal, saving your data and changes to your data to permenant storage.
Do not include the "" and please write in all lower case.
